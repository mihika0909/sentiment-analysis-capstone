# Sentiment Analysis System (Capstone Project)

## Project Overview
This project implements an NLP-based Sentiment Analysis System that classifies text as positive or negative. 
The system compares traditional machine learning models with deep learning and integrates sentiment detection into a simple chatbot.

---

## Dataset
- IMDB Movie Reviews Dataset
- Each review is labeled as **positive** or **negative**
- Dataset file: `sentiment.csv`

---
